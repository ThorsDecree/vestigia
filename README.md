Vestigia

Building continuity.

A deliberately small, hand-written website built with semantic HTML and CSS.

The site is treated as a living workshop rather than a finished product. It grows by addition, not replacement.

Vestigia Design Principles
- Calm before clever.
- Whitespace is hospitality.
- Typography carries emotion.
- Every page answers one question.
- Every section leads somewhere.
- Build by addition, not replacement.
- Leave good maps.
